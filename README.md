# tucfetch
## Dependencies
- curl
- wget
- neofetch
- chafa (or any other backend supported by neofetch)

## Install
Just install dependencies and run the `install.sh` script
```sh
cd tucfetch/
./install.sh
```

## Uninstall
Delete `~/.local/bin/tucfetch` and/or `/usr/bin/tucfetch`
You might also want to delete `~/.cache/tucfetch`
